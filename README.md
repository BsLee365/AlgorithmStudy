## 학습방법

1. 퇴근하고 나서 나동빈 알고리즘 강의 하루에 30분씩 듣는다.
2. 알고리즘 문제 한문제 풀고자기
3. 만약, 풀지 못했다면 회사에서도 천천히 생각해서 최대한 풀어보기
4. 그래도 풀지 못했다면, 공략을 보고 내 것으로 만든다.
